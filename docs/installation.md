# Install

## Prerequisites

`````{md-tab-set}

````{md-tab-item} Brew
```{code-block} console
brew install htslib
brew install openssl
```
````

````{md-tab-item} Conda
```{code-block} console
conda install htslib
conda install openssl
```
````

````{md-tab-item} Apt-get
```{code-block} console
apt-get update && apt-get install libhts-dev libssl-dev
```
````

`````
